# 生成浏览器水印


### usage

```javascript
import Watermark from '@iamgx/watermark'
Watermark({ text: '水印' });

```


### property

```javascript
import Watermark from '@iamgx/watermark'
Watermark({ 
  target: () => selector,
  text: '水印内容',
  width: 140,
  height: 80,
  color: "black",
  opacity: 0.2,
  zIndex: 999999999,
  font: "15px Arial",
});

```
